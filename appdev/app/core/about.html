About
Startpage focused on practical features rather then flashy look.

Practical Startpage is for those that want a functional Startpage, allowing easy access to chrome data, such as bookmarks and history. Right now there is only edit for bookmarks, but more is planned.

Once I have written interface specification for widgets, the code will be available on GitHub for anyone to add features they would like. Hopefully with Release 2.1...
